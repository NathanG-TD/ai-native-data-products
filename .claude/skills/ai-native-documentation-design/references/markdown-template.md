## Empty Section Convention

When a section has no records for the requested data product/module/version/date:

```markdown
## {Section Name}

No {items} recorded for {data_product} / {module_name} at version {version} / as of {date}.
```

For example:
```markdown
## Query Cookbook

No recipes recorded for customer_banking / GOLD at version 1.0.0 / as of 2026-02-28.
```
